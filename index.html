MBTI 직장인 성향 검사

환영합니다!

이 검사는 직장 내에서의 당신의 성향을 MBTI 방식으로 진단합니다.
각 상황에 대해 본인에게 더 가까운 행동을 선택해 주세요.
결과는 그래프와 해설로 제공됩니다.

시작하기 버튼을 눌러주세요!

외향(E) / 내향(I)
1. 동료를 처음 만났을 때 어떻게 행동하나요?
2. 회의에서 자신의 의견을 어떻게 표현하나요?
3. 점심시간에 보통 어떻게 식사하나요?
4. 사내 행사나 모임에 얼마나 자주 참여하나요?
5. 새로운 동료가 들어왔을 때 어떻게 하나요?
6. 팀 프로젝트를 시작할 때 어떻게 준비하나요?
7. 업무 중 도움이 필요할 때 어떻게 하나요?
8. 직장에서 대화하는 스타일은?
9. 새로운 모임에서 자신을 어떻게 소개하나요?
10. 동료의 일에 관심을 가지는 편인가요?
감각(S) / 직관(N)
1. 새로운 업무를 맡았을 때 어떻게 시작하나요?
2. 보고서 작성 시 중요하게 생각하는 것은?
3. 업무 방식에 대해 어떻게 생각하나요?
4. 문서 작성 시 어떤 부분을 더 강조하나요?
5. 업무 중 더 집중하는 것은?
6. 업무를 처리할 때 어떤 스타일인가요?
7. 아이디어 회의에서 어떤 아이디어를 주로 내나요?
8. 업무 계획을 세울 때 더 중시하는 것은?
9. 업무와 관련된 정보를 볼 때 더 중요하게 생각하는 것은?
10. 문제가 발생했을 때 더 집중하는 것은?
사고(T) / 감정(F)
1. 동료와 의견이 다를 때 어떻게 설득하나요?
2. 문제가 있을 때 어떻게 지적하나요?
3. 평가를 할 때 어떤 점을 더 고려하나요?
4. 결정을 내릴 때 더 중요하게 생각하는 것은?
5. 갈등이 생겼을 때 우선하는 것은?
6. 업무 처리 시 더 중시하는 것은?
7. 의견을 나눌 때 더 중요하게 생각하는 것은?
8. 동료가 고민을 털어놓을 때 어떻게 반응하나요?
9. 문제를 제기할 때 어떤 방식을 선호하나요?
10. 의사소통 시 더 중시하는 것은?
판단(J) / 인식(P)
1. 프로젝트 계획을 세울 때 어떤 방식을 선호하나요?
2. 업무 마감 일정에 대한 태도는?
3. 일정 및 할 일 관리 방법은?
4. 업무 진행 방식은?
5. 결정을 내릴 때 더 중요한 것은?
6. 업무 공간이나 자료 정리에 대한 태도는?
7. 업무 처리 순서는?
8. 일정 확인 및 결정 방식은?
9. 팀 내 역할 분담 방식은?
10. 예상치 못한 변화에 대한 태도는?
